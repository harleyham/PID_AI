2) Build principal multi-GPU na máquina com CUDA 12.8

Na máquina com P1000 + CUDA 12.8, use o código da árvore do projeto:

conda activate ham_P1000
cd /media/ham/EXT4/PROJETO_LIGEM_HIBRIDO/03_Scripts_Common/Src/pdal-cuda-plugin

rm -rf build
mkdir build
cd build

cmake .. \
  -G Ninja \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_PREFIX_PATH="$CONDA_PREFIX" \
  -DCMAKE_CXX_COMPILER=/usr/bin/g++ \
  -DCMAKE_CUDA_HOST_COMPILER=/usr/bin/g++ \
  -DCMAKE_CUDA_ARCHITECTURES="61;89;120"

ninja -j"$(nproc)"

Depois copie o .so para a pasta central de binários:
